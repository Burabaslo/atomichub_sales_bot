# atomichub_sales_bot
Telegram bot for notifications of the latest sales of items on Atomichub market

Для чего это нужно?

Бот создан для того, чтобы отслеживать продажи NFT предметов на площадке Atomichub.
Актуален в силу того, что график продаж для новых предметов на самой площадке строится спустя 5-10 минут, после их выпуска (дропа).
С ботом вы первым получаете информацию, о продажах заданного предмета.
Благодаря этому вы знаете за сколько его можно выставить (если он у вас есть) или за сколько его купить с площадки,
потому что в первые минуты после выпуска разброс цен бывает большим.

Как запустить?

Для запуска нужно скачать файлы и закинуть в config.py свой телеграм id и токен телеграм бота, после чего запустить bot.py
Для начала работы с ботом напишите ему в телеграм /start, он отправит пример json'а с настройками.
Заполните настройки под конкретный предмет, время проверки продаж, частоту проверки и время отправки уведомлений.
Готово :)
